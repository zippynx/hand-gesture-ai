# Hand Gesture Volume Control

Control your system volume using hand gestures in real time with Python, OpenCV, and MediaPipe.

## Features

- Real-time hand tracking
- Finger distance detection
- System volume control using gestures
- FPS counter
- Interactive volume bar UI
- Webcam-based AI interaction

## Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy
- Pycaw
